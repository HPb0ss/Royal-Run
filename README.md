# Royal-Run
A 3D infinite runner game with falling obstacles!

Royal Run is a 3D infinite runner game, dodge fences, rocks and falling obstacles as you move from checkpoint to checkpoint to increase the timer. Try to get as high of a score as possible, picking up coins and powerups along your way to increase the score.

## How To Play :-

Go to https://royal-run.vercel.app/
OR
To download the game, click on Game upload in releases and click on the Royal Run.zip, download the zip file, unzip it and run the Royal Run application, have fun!!!!!

Use WASD to move and dodge obstacles, hitting obstacles slows you down. Picking up coins increases the score and apples increase the move speed.

Every 8th Tile is a checkpoint which increases your the time remaining, the run ends when the timer hits 0.
